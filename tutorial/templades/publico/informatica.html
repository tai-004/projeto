{% extends 'publico.html' %} {% block content %} Curso Técnico em Informática para a Internet Iniciado em 2017, o Curso Técnico em Informática para a Internet ofertado pelo campus Guanambi destina-se àqueles alunos que já concluíram o Ensino Fundamental II, ou seja, o 9º ano. Numa perspectiva mais intimista, é o curso integrado mais novo nessa categoria, porém isso não o torna mais fácil. É realizado (assim como o nome sugere) de forma integrada em período diurno juntamente com o Ensino Médio tradicional. Após passar por uma pequena adequação do currículo, o curso é focado no desenvolvimento web (sim, desenvolvimento de sites e sistemas para navegador). Nele o discente aprende tanto sobre a área de redes de computadores ( a questão de cabeamento, por exemplo), há uma período sobre informática básica, porém o foco é em Lógica de Programação, HTML, CSS, Bootstrap, Javascript. Após, o discente começa a caminhar para áreas mais complexas, como a linguagem Python e framework Django, arquitetura de servidores e demais assuntos pertinentes. É o único curso em que desenvolver o PCC (Projeto de Conclusão de Curso) é obrigatório para a formação, não se esqueça do estágio. Muita coisa, não? {% endblock %}
